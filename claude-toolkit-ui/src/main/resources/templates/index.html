Quick Start — 로컬 실행
# 1. Clone & Build
git clone https://github.com/Sangmoo/Claude-Java-Toolkit.git
cd Claude-Java-Toolkit
mvn install -DskipTests

# 2. API 키 설정
export CLAUDE_API_KEY=sk-ant-...         # Mac/Linux
set CLAUDE_API_KEY=sk-ant-...            # Windows

# 3. UI 실행
cd claude-toolkit-ui
mvn spring-boot:run

# → http://localhost:8027 (설치 마법사 → 로그인: admin / admin1234 → 비밀번호 변경 필수)
Quick Start — Docker 배포
# 1. 환경변수 설정
cp .env.example .env     # .env에서 CLAUDE_API_KEY 설정

# 2-A. H2 DB (기본, 별도 DB 불필요)
docker-compose up -d claude-toolkit

# 2-B. MySQL 사용 시
DB_TYPE=mysql docker-compose --profile mysql up -d

# 2-C. PostgreSQL 사용 시
DB_TYPE=postgresql docker-compose --profile postgresql up -d

# → http://localhost:8027
헬스체크: /actuator/health · H2 / MySQL / PostgreSQL · 초기 계정: admin / admin1234 (첫 로그인 시 변경 필수)
