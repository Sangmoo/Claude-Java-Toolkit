Claude Java Toolkit 설치 마법사

4단계로 간단하게 설정을 완료하세요

Step 1. Claude API 키
Anthropic Claude API 키를 설정합니다. 환경변수(CLAUDE_API_KEY)로도 설정 가능합니다.
API 키가 이미 설정되어 있습니다.
API 키가 설정되지 않았습니다. 환경변수 CLAUDE_API_KEY를 설정하세요.
Step 2. 데이터베이스
애플리케이션 데이터 저장용 DB를 선택합니다. 기본값은 H2 파일 DB입니다.
H2 파일 DB (기본값) — 별도 설정 없이 사용 가능
MySQL/PostgreSQL로 전환하려면 환경변수 DB_TYPE=mysql 또는 DB_TYPE=postgresql을 설정하세요.
Step 3. 이메일 (선택)
분석 결과 이메일 발송을 위한 SMTP를 설정합니다. 나중에 Settings에서 설정할 수 있습니다.
Settings 페이지에서 SMTP 서버, 포트, 계정을 설정하면 이메일 발송이 활성화됩니다.
Step 4. 설치 완료
기본 설정이 완료되었습니다.
기본 관리자 계정: admin / admin1234
추가 설정은 Settings 페이지에서 변경할 수 있습니다.
보안을 위해 비밀번호를 반드시 변경하세요.
