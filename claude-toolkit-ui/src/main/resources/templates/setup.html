Claude Java Toolkit 설치 마법사

4단계로 간단하게 설정을 완료하세요

Step 1. Claude API 키
Anthropic Claude API 키를 입력하고 연결 테스트 후 저장합니다.
API 키가 이미 설정되어 있습니다.
Step 2. Oracle DB 연결 (선택)
SQL 분석 대상 Oracle DB를 설정합니다. 나중에 Settings에서도 설정 가능합니다.
내부 데이터 저장용 DB(H2)는 별도 설정 없이 자동 사용됩니다.
MySQL/PostgreSQL로 전환하려면 환경변수 DB_TYPE=mysql|postgresql 설정 후 재시작하세요.
Step 3. 이메일 설정 (선택)
분석 결과 이메일 발송을 위한 SMTP를 설정합니다. 나중에 Settings에서도 설정 가능합니다.
Gmail 사용 시 "앱 비밀번호"를 생성하여 입력하세요. 이 단계를 건너뛰어도 됩니다.
Step 4. 설치 완료
기본 설정이 완료되었습니다.
기본 관리자 계정: admin / admin1234
추가 설정은 Settings 페이지에서 변경할 수 있습니다.
보안을 위해 비밀번호를 반드시 변경하세요.
