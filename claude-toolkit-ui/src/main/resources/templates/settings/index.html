Oracle DB 연결 정보, Java 프로젝트 경로, 프로젝트 컨텍스트를 설정합니다.

설정이 저장되었습니다.
Claude API 연결
API 키는 환경변수 CLAUDE_API_KEY 또는 application.yml로 설정합니다.
Oracle DB 연결
예: jdbc:oracle:thin:@//192.168.1.100:1521/ORCL
DB 설정됨 미설정
Java 프로젝트 경로
Controller, Service, Repository, VO/DTO 파일이 자동 분류됩니다. target/, test/는 자동 제외됩니다.
경로 설정됨 미설정
프로젝트 컨텍스트 메모 설정됨

프로젝트 개요, 기술 스택, 코딩 컨벤션 등을 입력하면 모든 AI 요청에 컨텍스트로 자동 추가됩니다.

최대 권장: 2,000자. 이 내용은 각 기능의 AI 분석 품질을 향상시킵니다.
Claude 모델 선택

기본값은 application.yml에 설정된 모델입니다. 여기서 선택하면 런타임에 즉시 반영됩니다.

현재 사용 중인 모델:
취소 저장 위치: ~/.claude-toolkit/settings.json (재시작 후에도 유지됨)
환경변수로 영구 설정
# Claude API
export CLAUDE_API_KEY=sk-ant-...

# Oracle DB
export ORACLE_DB_URL=jdbc:oracle:thin:@//hostname:1521/SERVICE_NAME
export ORACLE_DB_USERNAME=myuser
export ORACLE_DB_PASSWORD=mypassword

# Java 프로젝트 경로
export PROJECT_SCAN_PATH=C:/workspace/my-spring-project/src/main/java
