Claude 모델 선택

기본값은 application.yml에 설정된 모델입니다. 여기서 선택하면 런타임에 즉시 반영됩니다.

현재 사용 중인 모델:
테마 액센트 색상

UI 전체의 강조 색상(버튼, 링크, 아이콘 등)을 원하는 컬러로 변경합니다.

현재 색상: [[${settings.accentColor != null and !settings.accentColor.isEmpty()} ? ${settings.accentColor} : '#f97316']]
기본값: #f97316 (오렌지) — 저장 시 즉시 적용됩니다.
이메일 알림 (SMTP)

스케줄 작업 완료 시 결과를 이메일로 발송합니다. Gmail은 앱 비밀번호를 사용하세요.

이메일 설정 완료 — 스케줄 작업에서 수신 주소를 입력하면 발송됩니다.
취소 저장 위치: ~/.claude-toolkit/settings.json (재시작 후에도 유지됨)
환경변수로 영구 설정
# Claude API
export CLAUDE_API_KEY=sk-ant-...

# Oracle DB
export ORACLE_DB_URL=jdbc:oracle:thin:@//hostname:1521/SERVICE_NAME
export ORACLE_DB_USERNAME=myuser
export ORACLE_DB_PASSWORD=mypassword

# Java 프로젝트 경로
export PROJECT_SCAN_PATH=C:/workspace/my-spring-project/src/main/java
