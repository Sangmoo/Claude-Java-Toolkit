코드 리뷰 하네스 4단계 탭
Analyst → Builder → Reviewer → Verifier 4단계 AI 파이프라인이 한 화면에 통합되지 않고 탭별로 분리되어 표시됩니다. 활성 단계가 자동으로 탭 전환되며, 각 단계와 전체 결과 모두 복사·MD·PDF 인쇄·이메일 발송이 가능합니다. 백엔드는 sentinel 마커로 단계 경계를 명확히 표시하고, 청크 파서가 chunk 경계 분할까지 안전하게 처리합니다.
🔍 Analyst 🔧 Builder 📋 Reviewer 🛡️ Verifier 📑 단계별 발송
🔁
분석 파이프라인 오케스트레이터
YAML 또는 비주얼 빌더로 N 단계 분석을 조합하고 SSE 로 실시간 진행률을 확인합니다. 단계 간 컨텍스트 전파가 보장되고, max_tokens 도달 시 chatStreamWithContinuation 으로 자동 이어받기 (단계당 최대 32K 토큰), 클라이언트 disconnect (Broken pipe) 시에도 백그라운드 실행이 중단되지 않으며, 모든 결과는 자동으로 리뷰 이력에 1건으로 저장됩니다.
🔗 단계 간 컨텍스트 🔄 자동 이어받기 📝 이력 자동 저장 ⚡ 실시간 SSE
⚙️
사용자 권한 동적 토글
기능 목록을 백엔드 한 곳에서만 관리하고 (AdminPermissionController.FEATURES), 사용자 권한 관리 페이지가 GET /admin/permissions/features 로 동적 로드합니다. 항목 추가 시 동기화 누락 버그가 구조적으로 차단되고, 로그인 직후부터 사이드바가 권한 토글을 즉시 반영합니다 (이전엔 새로고침 후 적용).
🔐 동적 권한 🚀 즉시 반영 🧩 단일 정의
🛠️
사내망 운영 안정화
Ubuntu 기반 JRE 이미지로 전환하고 TLSv1.2/1.3 + DNS 캐시 + Forward Proxy 지원을 추가해 사내망 TLS handshake_failure 를 해결했습니다. Oracle ORA-01882 timezone 이슈는 timezoneAsRegion=false + TZ=Asia/Seoul 로 영구 해결. 자동 이관 후 런타임 DB 전환 + 시스템 헬스 실제 DB 반영 + 분석 결과 이메일 발송(최대 10명) + Gmail SMTP 설정 UI 까지 운영 기능이 대폭 보강되었습니다.
🔐 TLS 1.2/1.3 🌐 Forward Proxy 🔄 런타임 DB 전환 ✉️ 다수 수신자 메일

시작하기

5분 만에 시작하기

JDK 1.8 이상과 Claude API 키만 있으면 바로 시작할 수 있습니다

1
레포지토리 클론
Git으로 레포지토리를 클론하고 Maven으로 빌드합니다. JDK 1.8+, Maven 3.6+이 필요합니다.
git clone https://github.com/Sangmoo/Claude-Java-Toolkit.git
cd Claude-Java-Toolkit
mvn package -DskipTests
2
실행
빌드된 JAR을 실행하면 포트 8027에서 서버가 시작됩니다.
java -jar claude-toolkit-ui/target/claude-toolkit-ui-*.jar
# → http://localhost:8027
3
Claude API 키 설정
Settings 페이지에서 Anthropic API 키를 입력합니다. 키는 로컬 파일에 암호화 없이 저장되므로 개인 PC에서 사용하세요.
Settings → Claude API Key → 저장
# ~/.claude-toolkit/settings.json 에 저장됩니다
4
Oracle DB 연결 (선택)
실행계획 분석, ERD 자동 스캔이 필요한 경우 Settings에서 Oracle DB 연결 정보를 등록합니다. 없어도 SQL·코드 리뷰는 사용 가능합니다.
Settings → Oracle DB 연결
JDBC URL: jdbc:oracle:thin:@//host:1521/ORCL
5
분석 시작
사이드바에서 원하는 기능을 선택하고 SQL 또는 코드를 붙여넣기 후 분석을 실행합니다. 결과는 Markdown으로 내보낼 수 있습니다.
